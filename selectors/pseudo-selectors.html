Advanced selectors

Pseudo-Class Selectors

pseudo selectors are keywords added after a selector with a colon symbol.

Change the color when hovering over a link.

a:hover {
  color: green;
}

Hover over this link to see it change to green.

Pseudo Element Selectors

pseudo element selectors select certain parts of element that are not part of the HTML.

pseudo element selectors use double colon notation to differentiate between peseudo class selectors.

NOTE: Older versions of CSS use a single colon. Modern browser will work with either.

::before and ::after

:before and :after used in box-model-fix. They are also used to GENERATE elements inserted before or after the selected element

These generated elements do NOT become part of the HTML DOM.

These must always be used with the content property.

.someClass::before {
  content: "words for the pseudo selector ";
}

This is the normal text in the HTML.

Using ::before and ::after to add quote marks

If you need to put quotes around quoted text, you just need to escape the double quote.

You could also put the double quote inside of single quotes , or use unicode '\201C'

.someClass::before {
  color: red;
  content: "\"";
}

.someClass::after {
  color: red;
  content: "\"";
}

Red quotes are added.
