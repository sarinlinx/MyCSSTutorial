CSS selectors

This page explains what CSS selectors are and how to use them.

A CSS selector locates and HTML element and adds styles to it.

Syntax

The basic syntax is: SELECTOR { PROPERTY: VALUE }

CSS can locate HTML elements based on their:

  • element name
  • id
  • class
  • attribute

The 'Element' selector

The 'Element' selector locates an HTML element based on the element name. Examples include:

  • body
  • h1
  • p

Here's an example of a paragraph 'Element' selector.

This Paragraph selector would locate all paragraphs in the document.

p {
  color: blue;
}

The 'ID' selector

The 'ID' selector locates an HTML element based on a specific ID value given to that element. For example:

Assign a paragraph the 'id' of 'intro'.

<p id="intro">words words words</p>

You can then target it in your CSS file with a #.

#intro {
  background-color: blue;
}

You can either use single quotes or double quotes around the 'id' name.

The 'Class' selector

The 'Class' selector locates an HTML element based on a specific Class name value given to that element. For example:

Assign a paragraph the 'Class' of 'center'.

<p class="center">words words words</p>

You can then target it in your CSS file with a period.

.center {
  background-color: blue;
}

You can either use single quotes or double quotes around the 'Class' name.

Grouping selectors together

You can write a single declaration for multiple selectors are once. Just separate them with a comma.

This CSS applies to h1, h2, and p tags.

h1, h2, p {
text-align: center;
color: red;
}

Multiple classes can be grouped together.

This matches the HTML element ONLY IF both classes are set in it.

<p class="one two" id="three"></p>
                  .one.two {
/*styles*/
}

Specifying element selectors

You can specifically assign a class to only one HTML element using a period to separate them.

This CSS applies only to h1 elements with a class of 'center'.

h1.center {
  text-align: center;
}

CSS Combinators

Combinators explain the relationship between the selectors. Between the selectors, you can add a combinator.

Examples of combinators:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descendant selectors

Descendant selectors add a space between the selectors. This makes it possible to match all elements that are descendants of a specified element.

This example looks inside all <div> elements and selects any <p> elements inside. 
div p {
  background-color: yellow;
}
You can also skip levels. This example looks inside all <ancestor> elements and selects any <descendant> elements inside. 
<div class="ancestor">
  <div class="parent">
    <div class="child">This is not styled</div>
    <div class="descendant">This IS styled</div>
  </div>
</div>
CSS to select the descendant. 
.ancestor .descendant {
  color: blue;
}
Result: 
This is not styled
This IS styled
Here is another example that looks inside any <section> elements for <a> elements. 
<section>
  <div class="example">
    <p>Link goes here <a href="#">link</a>.</p>
  </div>
</section>
CSS to select the anchor tag within the section tag. 
section a {
  /*styles*/
}

Child selectors

Select ONLY the child element using > symbol between parent and child elements.

This example ONLY looks in the <section> element for a direct <a> element. 
<section>
  <a href="#">child 1</a>
  <a href="#">child 2</a>
    <div>
        <a href="#">descendant</a>
    </div>
</section>
CSS to select the first two anchor tag children inside the section tag. The 3rd isn't a child so it's not select. 
section > a {
  color: green;
}
Result: 
child 1
child 2
descendant

Adjacent Sibling Selectors

Adjacent Sibling Selectors only select the element immediately following a specified element. They are separated with a + symbol.

This example ONLY selects the paragraph element directly after an <h2> element. 
<p>Paragraph 1</p>

<h2>Heading</h2>
<p>Paragraph 2</p>
<p>Paragraph 3</p>
CSS to select the paragraph immediately following the h2 tag. The 3rd doesn't immediately follow so it's not selected. 
h2 + p {
 color: blue;
}
Result: 
Paragraph 1
Heading
Paragraph 2
Paragraph 3

General Sibling Selector

General Sibling Selectors select all specified elements following a specified element. They are separated with a ~ symbol.

This example selects all paragraphs following the <div> element. 
<p>Paragraph 1.</p>
<div>
  <p>Paragraph 2.</p>
</div>
<p>Paragraph 3.</p>
<p>Paragraph 4.</p>
CSS to select only the paragraphs that follow the div tag. 
div ~ p {
  color: red;
}
Result: 
Paragraph 1
Paragraph 2
Paragraph 3
Paragraph 4
