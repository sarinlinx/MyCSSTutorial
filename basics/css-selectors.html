CSS selectors

This page explains what CSS selectors are and how to use them.

A CSS selector locates and HTML element and adds styles to it.

Syntax

The basic syntax is: SELECTOR { PROPERTY: VALUE }

CSS can locate HTML elements based on their:

  • element name
  • id
  • class
  • attribute

The 'Element' selector

The 'Element' selector locates an HTML element based on the element name. Examples include:

  • body
  • h1
  • p

Here's an example of a paragraph 'Element' selector.

This Paragraph selector would locate all paragraphs in the document.

p {
  color: blue;
}

The 'ID' selector

The 'ID' selector locates an HTML element based on a specific ID value given to that element. For example:

Assign a paragraph the 'id' of 'intro'.

<p id="intro">words words words</p>

You can then target it in your CSS file with a #.

#intro {
  background-color: blue;
}

You can either use single quotes or double quotes around the 'id' name.

The 'Class' selector

The 'Class' selector locates an HTML element based on a specific Class name value given to that element. For example:

Assign a paragraph the 'Class' of 'center'.

<p class="center">words words words</p>

You can then target it in your CSS file with a period.

.center {
  background-color: blue;
}

You can either use single quotes or double quotes around the 'Class' name.

Grouping selectors together

You can write a single declaration for multiple selectors are once. Just separate them with a comma.

This CSS applies to h1, h2, and p tags.

h1, h2, p {
text-align: center;
color: red;
}

Specifying element selectors

You can specifically assign a class to only one HTML element using a period to separate them.

This CSS applies only to h1 elements with a class of 'center'.

h1.center {
  text-align: center;
}

Attribute selectors

The attribute selector locates elements with a specified attribute. Here are a few examples.
