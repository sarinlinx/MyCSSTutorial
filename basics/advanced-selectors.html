Advanced selectors

Descendent selectors

Use multiple selectors, separated by a space to match the descendent elements.

Example: Select a link in a paragraph and change the color to red.

<p>Please click <a href="#">HERE!!!</a></p>

p a {
  color: red;
}

Multiple descendent selectors

Select paragraphs and links inside <section>

<section>
<p>Please click <a href="#">HERE!!!</a></p>
</section>

section p a {
  color: blue;

Grouping selectors

Grouping selectors that share the same properties helps to simplify code.

This will make the text for all h1, h2, and h3 tags red.

h1, h2, h3 {
  color: red;
}

Pseudo-Class Selectors

pseudo selectors are keywords added after a selector with a colon symbol.

Change the color when hovering over a link.

a:hover {
  color: green;
}

Hover over this link to see it change to green.

Selecting first and last children of parent elements

Apply a pseudo selector to a child element of a parent to specifically target that child.

This must be added to the child element.

Select the first and last child elements from the parent.

NOTE how the first-child and last-child selectors are added to the child elements, not the parent.

<section>
  <p>Paragraph</p> <!-- this is red -->
  <p>Paragraph</p>
  <p>Paragraph</p> <!-- this is blue -->
</section>

p:first-child {
  color: red;
}
p:last-child {
  color: blue;
}



  Paragraph <!-- this is red -->
  Paragraph
  Paragraph <!-- this is blue -->

Selecting first and last type from a parent element.

This is different from first-child last-child since it doesn't matter what order the HTML is in.

The first type of the specified element will be selected.

Select the first and last paragraph elements in a parent.

Notice how the paragraph tags are not first or last inthe HTML, but this pseudo element allows you to target them since they are the first and last of that specific type of element.

<section>
  <h1>heading</h1>
  <p>Paragraph</p> <!-- this is red -->
  <p>Paragraph</p>
  <p>Paragraph</p> <!-- this is blue -->
  <a href="#">link</a>
</section>

p:first-of-type {
  color: red;
}
p:last-of-type {
  color: blue;
}
