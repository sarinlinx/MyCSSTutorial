Comments and shortcuts

Recommended comment sections

There are a few standards to use when adding comment sections to your CSS code.

For example, if you have an entire section to style 'lists', you should write comment block above so it's easy to view what that block is about:

/*____________________
Lists
____________________*/

ul {
  /* styles */
}
