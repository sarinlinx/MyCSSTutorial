CSS backgrounds

This page explains how to add backgrounds to your web page.

Adding general backgrounds

The following shows how to add a general color, gradient, or image to the background.

Setting a background color

Add a single color background.

body {
  background-color: lightblue;
}

Setting a background image

Add a background image.

body {
  background-image: url("paper.gif");
}

To specify multiple images, separate the URLs with a comma.

body {
  background-image: url("image1.png"),
  url("image2.png");
}

Background transparency

If there is already a background added, you can add a second layer over it. You can then adjust this second layer's transparency value.

Make sure this is NO SPACE between rgba and the parenthese.

<div class="image-bg">
  <p class="rgba-transparency">Testing to see how the transparency works.</p>
</div>

.image-bg {
  background: url("images/note.jpg")
}

.rgba-transparency {
  background: rgba(199, 21, 133, .25);
}

The result will be a transparent layer over top the current background.

Linear gradient background

This shows how to add a background of 2 colors that blend into each other.

<p class="linear-gradient">Testing to see how the linear gradient works.</p>


.linear-gradient {
  background: linear-gradient(red, green)
}

The result will be a transparent layer over top the current background.

You can also add a 3rd color.

<p class="linear-gradient-3colors">Testing to see how the linear gradient works.</p>


.linear-gradient-3colors {
background: linear-gradient-3colors(red, yellow, green),
}

The result will be a transparent layer over top the current background.

Transparent linear gradient over a background image

This shows how to add a transparent linear gradient of 3 colors over a background image.

The linear-gradient values must come first.

<div class="image-bg">
  <p class="transparent-linear-gradient">Transparent linear gradient over an background image.</p>
<div>


.transparent-linear-gradient {
  height: 40px;
  background: linear-gradient(rgba(255, 0, 0, 0.35),  /*red*/
  rgba(255, 255, 0, 0.35), /*yellow*/
  rgba(0, 0, 255, 0.35)),  /*blue*/
  url("images/backgrounds/examples/white-bg.jpg");
}

The result will be a transparent gradient over a background image.

Background repeat

The following shows different ways to repeat the background image.

By default, the background-image property repeats an image both horizontally and vertically.

Repeat the image on the x axis.

This repeats the image horizontally.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat: repeat-x;
}

The image is repeated horizontally only. If the image has a short height, anything below is inherits its background color.

Repeat the image on the y axis.

This repeats the image vertically.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat: repeat-y;
}

The image is repeated horizontally only. If the image has a short width, anything to the right inherits its background color.

Do not repeat the image.

This forces the image to not repeat at all.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat: no-repeat;
}

The result is that the image shows once in the top left corner.

Background attachement

By default, the background image will not move with the page.

The background attachment property allows you to control what happens to the background image as the page is navigated.

Fixed — Fix the image to the page as you scroll.

This causes the image to be fixed to the top left of the page. As the user scrolls, the image always remains on the top left and never goes out of sight.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat: no-repeat;
  background-attachment: fixed;
}

This only works on the body element. If you used 'fixed' for any other element, the image will disappear. This is because 'fixed' is relative to the viewport, not the element you're adding the code to.

Scroll — Allow the image to scroll with the page.

This causes the image to scroll with the page. As the user scrolls, the image scrolls also. This is the default behavior.

body {
background: url("../images/backgrounds/examples/purple-dot.jpg");
background-repeat: no-repeat;
background-attachment: scroll;
}

This causes the image to scroll with the page. As the user scrolls, the image scrolls also. This is the default behavior.

Background Position

Position the background image to a specific area of the page.

Using positional words

You can use words like 'top right' to position the background.

body {
background: url("../images/backgrounds/examples/purple-dot.jpg");
background-repeat: no-repeat;
background-position: top right;
}

The result will be the image on the top right.

Using % values

You can use % values to position the background.

The first value is the horizontal position and the second value is the vertical. The top left corner is 0% 0%. The right bottom corner is 100% 100%.

If you only specify one value, the other value will be 50%. Default value is: 0% 0%.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat: no-repeat;
  background-position: 20% 10%;
}

This example moves the image 20% to the right and 10% down on the page.

You can position using any CSS unit like pixels.

This example moves the image 20 pixels to the right and 50 pixels down on the page.

Background size

Background size specifies the size of background images.

There are four different syntaxes you can use with this property:

  • Keyword syntax ("auto", "cover" and "contain")
  • One-value syntax (sets the width of the image (height becomes "auto")
  • Two-value syntax (first value: width of the image, second value: height)
  • Multiple background syntax (separated with comma).

Keyword syntax

auto is the default value. The background image is displayed in its original size.

body {
  background: url("../images/backgrounds/examples/purple-dot.jpg");
  background-repeat:no-repeat;
  background-size:auto;
}

One value syntax

If you set only one value for width, the second value for height becomes 'auto'.

body {
background: url("../images/backgrounds/examples/purple-dot.jpg");
background-repeat:no-repeat;
background-size: 80px;
}

Example: The image is set to 80px.

You can also use percentage, but this is not recommended. This isn't a good idea because the aspect ratio does not change, so there could be blank space at the bottom of the page after the image.

Different background images for different displays

If you have a high resolution display (retina) you may want to display a different image than a smaller screen with less resolution.

Use a media query.

header {
  background: url(image.jpg);
}

@media (min-resolution: 192dpi) {
  header {
    /* Using a larger image for retina screens */
    background: url(image@2x.jpg);
  }
}
