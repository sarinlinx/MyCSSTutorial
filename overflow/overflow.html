The Overflow property

The overflow property controls what happens to content that is too big to fit into an area.

Possible values are:

  • visible - Default. The overflow is not clipped. It renders outside the element's box
  • hidden - The overflow is clipped, and the rest of the content will be invisible
  • scroll - The overflow is clipped, but a scrollbar is added to see the rest of the content
  • auto - If overflow is clipped, a scrollbar should be added to see the rest of the content

The overflow property only works for block elements with a specified height.

In OS X Lion (on Mac), scrollbars are hidden by default and only shown when being used (even though "overflow:scroll" is set).

You can also use white-space or text-wrap to change how text is displayed in the box.

visible

visible (default) - nothing is clipped. All text is 'visible'.

<p>Lots of text Lots of text Lots of text Lots of text
blah blah blah yes yea eyahd asdlkjfd adkfjd kajd lfadkj fla djfal fad</p>

p {
  width: 200px;
  height: 50px;
  background-color: #eee;
  overflow: visible;
}

scroll - Any overflowing text is clipped, but a scrollbar is added to scroll inside the box.

overflow-x and overflow-y

These properties specify whether to change the overflow of content just horizontally or vertically (or both).

These are NOT used with the 'overflow' propety (which controls x and y). These are used on their own, or with each other.

You can use visible, hidden, scroll, and auto with these also.

This uses white-space: nowrap so you can see if the scroll bar appears horizontally.

X is set to scroll while Y is hidden.

<p>Line 1 Line 1 Line 1 Line 1 Line 1 Line 1 Line 1 Line 1 Line 1
    Line 2 Line 2 Line 2 Line 2 Line 2 Line 2 Line 2 Line 2 Line 2
    Line 3 Line 3  Line 3  Line 3  Line 3  Line 3  Line 3  Line 3  Line 3</p>

p {
  width: 200px;
  height: 50px;
  background-color: #eee;
  overflow-x: scroll;
  overflow-y: hidden;
  white-space: nowrap;
}
