Make a menu horizontal using the 'display' property

Wrap the menu in a <nav> tag with a class name.

You can then pinpoint the anchor tags within it.

<nav class="display">
<a href="#">About</a>
<a href="#">Contact</a>
</nav>


.display a {
  padding: 10px; //add size to link
}

Example:


AboutContact

Notice how the greenbackground does not expand to fit the padding of the links. This is default behavior for inline elements. To fix, just make the anchor links inline-block.

For example:

.display a {
  padding: 10px; //add size to link
  display: inline-block; //to expand parent element with child
}

Example:


        AboutContact

Notice how there is a space between the anchor tags? That is because the HTML for each link is on a different line.

Instead of changing your HTML, just change the font to zero for the parent element, then add it for the child element.

Set parent font to zero.

Set child font (otherwise it would also be zero since it would inherit the parent font).

<nav class="display">
<a href="#">About</a>
<a href="#">Contact</a>
</nav>

.display {
  text-align: center;
  font-size: 0;
}

.display a {
  padding: 10px;
  display: inline-block;
  font-size: 16px;
}

Make a menu horizontal using the 'float' property

Using display:inline-block can be achieved with float, but floating requires you to clear the element after the floating boxes. So inline-block is easier.

Also, when using float, the display property has no affect.
