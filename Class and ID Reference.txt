Class and ID Reference
----------------------

.codebox - preformatted box that contains the code
.code-container - container that holds the code example
.example-container - text to left of code example
.example-container - Contains two <div>. One for text to the left and one for code examples to the right.
.main-nav - <ul> primary navigation section
.sub-nav - <ul> topics that display when main menu item is clicked on


Common code to cut & paste
--------------------------
<div class="example-container">
    <div class="example-text">
        <p>Test goes here</p>
</div>
  <pre class="codebox"><span class="code-html">CODE Here</span></pre>
</div>
